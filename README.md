# 🚑 TriageFlow – AI Emergency Room Engine

## 🧠 Overview

TriageFlow is a real-time system that intelligently prioritizes emergency room patients based on severity, waiting time, and risk, ensuring critical cases are treated first.

## ⚡ Features

* 🔴 Smart Priority Queue (dynamic patient ranking)
* 🖥️ Live Dashboard with real-time updates
* 🏥 Automatic Bed Allocation (no overbooking)
* 🚑 Ambulance Pre-Arrival with ETA tracking
* 🚨 MCI Disaster Mode (emergency prioritization)
* 🤖 AI Insights & Alerts
* 🔍 Explainable patient ranking

## 🛠️ Tech Stack

React / HTML / CSS / JavaScript
